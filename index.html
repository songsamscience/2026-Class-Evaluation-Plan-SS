지식/논술 팽창 데이터를 분석하여 미래를 예측하고, 탐사의 당위성을 과학적 근거로 논리적으로 서술함. 증거를 바탕으로 제안서를 작성하였으나, 데이터 심층 해석이나 논리적 타당성이 일부 부족함. 기본 지식은 포함되었으나, 탐사의 효용성을 뒷받침할 과학적 데이터 근거가 빈약함. 탐사의 목적이나 필요성을 과학적 근거 없이 감성적이고 단편적으로만 서술함. 기초 개념 서술이 누락되었으며, 제안서 형태를 갖추기 위해 전적인 도움이 필요함. 60
과정/데이터 천문 포털에서 목적에 맞는 데이터를 직접 추출하고, 신뢰성을 검토하여 정확하게 인용함. 지정된 프로그램을 활용하여 적절한 데이터를 수집하고 기본적인 출처 규칙을 준수함. 일반 포털 검색에 주로 의존하였고, 객관적인 천문 데이터 활용이나 출처 명시가 부족함. 수집 절차를 따르는 데 어려움을 겪으며, 타인의 데이터를 무분별하게 차용함. 수집 활동 자체에 참여하지 않거나 디지털 윤리 의식이 결여되어 지도가 필수적임. 40
